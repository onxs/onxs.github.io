  1. з©әзҷҪеӨ„еҸіеҮ» Import йҖүжӢ© General йҮҢзҡ„ Existing Projects into WorkspaceвҶ’NextпјҢBrowse жүҫеҲ°иҰҒеҜје…ҘйЎ№зӣ®жүҖеңЁпјҢйҖүдёӯиҰҒеҜје…Ҙзҡ„йЎ№зӣ®зӮ№еҮ»зЎ®е®ҡпјӣдёҖе®ҡиҰҒеӢҫйҖү Copy projects into eorkspaceвҶ’Finishпјӣ

  2. еҜје…Ҙжң¬ең°йЎ№зӣ®д№ӢеҗҺе…ҲиҝӣиЎҢй…ҚзҪ®пјҢй…ҚзҪ®жӯҘйӘӨеҰӮдёӢпјҡ

    (1). еҜје…ҘйЎ№зӣ® еҸіеҮ» mavenвҶ’update project вҶ’OKпјӣ

    (2). йЎ№зӣ®еҗҚвҶ’еҸіеҮ»вҶ’properties

  1. еҜје…Ҙж•°жҚ®еә“ж•°жҚ®жӯҘйӘӨеҰӮдёӢ:

    (1). еңЁ localhost еҲӣе»әдёҖдёӘж•°жҚ®еә“пјҢеҗҚеӯ—дёҺеҜје…Ҙж–Үд»¶дёҖиҮҙпјӣ

    (2) еҸіеҮ»йҖүжӢ©иҝҗиЎҢ SQL ж–Үд»¶пјӣж•°жҚ®еә“дёӯеҜје…Ҙж•°жҚ®ж–Үд»¶пјҢйҖүжӢ©еҘҪиҰҒеҜје…Ҙзҡ„ж–Үд»¶пјҢдёҚеӢҫйҖү вҖңжҜҸдёӘиҝҗиЎҢдёӯиҝҗиЎҢеӨҡйҮҚжҹҘиҜўвҖқ е’Ң вҖңSETAUTOCOMMIT=0вҖқпјӣејҖе§ӢеҚіеҸҜ

  2. еҜје…Ҙд№ӢеҗҺе°Ҷиө„жәҗж–Үд»¶дёӯе…ідәҺж•°жҚ®еә“зҡ„й…ҚзҪ®з”ЁжҲ·еҗҚеҜҶз Ғж”№жҲҗйңҖиҰҒзҡ„еҚіеҸҜпјӣ
